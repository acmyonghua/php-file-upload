(mixed)saveFile([gdcFileUploaderFile $file = null], [string $fileName = null], [string $uploadDir = null])

Moves an uploaded file to a new location and renames it

ARGUMENTS
$file A gdcFileUploaderFile instance; leave empty if only one file was uploaded
$fileName The new file name; May lead to data loss!
$uploadDir The upload directory; is set as new global uploadDir

returns (mixed) (string) the file name, if success, otherwise (bool) false

throws (gdcFileUploaderException) gdcFileUploaderException

(bool)saveAllFiles([string $uploadDir = null], [mixed $callable = null])

ARGUMENTS

returns (bool) true, if there were no errors, otherwise false

throws (gdcFileUploaderException) gdcFileUploaderException
